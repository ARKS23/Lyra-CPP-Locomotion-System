# Lyra C++ Locomotion System
本项目基于UE-5.7，使用C++复刻Lyra Starter Game的运动系统。

## 阶段性目标
- 掌握Lyra的运动系统架构设计
- 将Blueprint中数据获取和计算逻辑用C++重构
- 构建模块化、可拓展的角色运动系统
